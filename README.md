# Sign_language

## 프로젝트 소개
"수어를 이용하여 ChatGPT를 사용할 수 있는 서비스"

## 개발 기간 및 인원
- 2023.05.17(수) - 2023.05.23(금)
- 개발 인원 : 7명
- 딥러닝 모델 개발 및 MLflow 기반 모델 관리
- 장고 서버 구축(AWS), ChatGPT API 사용
- 웹 디자인 (Front-End)
